Polls App Project

Description: 
  This project is for creating and viewing polls submitted by users. Each user can login and see the polls they have answered and the polls they have not answered. Each user can also create a new poll in which other users can answer. In addition there is a stats page which gives each user a score based on how many polls have been created and answered.

Installation: 
  Step 1: Clone the source code
    Run the command "git clone https://github.com/Fivemindz/polls-project.git"

  Step 2: Install dependencies
    Navigate to the cloned repository and run the commmand "npm install"

  Step 3: Start the dev server
    From the cloned repository run the command "npm start"
  
Testing:
  A testing suite is included in this repository. To run the test use the command "npm test"
